
# Maternal Health Data Analysis Using SQL

## Project Overview

This project analyzes maternal mortality data using PostgreSQL to identify mortality trends, regional patterns, population-adjusted health indicators and leading causes of reported deaths. The analysis provides insights that can support evidence-based public health planning and decision making.

## Objectives

- Explore maternal health records.
- Analyze mortality and health indicator trends.
- Compare disease burden across regions and states.
- Evaluate population-adjusted rates.
- Identify leading causes of reported deaths.

## Key Questions Answered

1. What is the total number of reported events?
2. Which region recorded the highest number of reported cases?
3. Which states contributed the most reported cases?
4. What is the population-adjusted rate per 100,000 people?
5. Which states recorded the highest indicator values?
6. How are reported cases distributed across Nigeria's geopolitical zones?
7. How have case rates changed over the years?
8. Does population size influence the number of reported deaths (population impact)?
9. Which health indicator (Maternal Death, Neonatal Death, or TB Death) recorded the highest number of deaths in each state?

## Key Insights

- Identified regions and states with the highest disease burden.
- Compared mortality using population-adjusted rates.
- Evaluated yearly trends in reported cases.
- Assessed the relationship between population size and reported deaths.
- Identified the leading cause of death within each state.
